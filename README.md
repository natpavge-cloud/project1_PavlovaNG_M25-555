# project1_PavlovaNG_M25-555
Лабиринт Сокровищ - текстовый Квест

1. Описание проекта

Лабиринт Сокровищ - это классическая текстовая приключенческая игра, написанная на Python. Игрок исследует таинственный лабиринт, собирает предметы, решает загадки и пытается найти легендарные сокровища, спрятанные в самой глубине подземелий.

Особенности игры:
• Текстовая навигация - перемещение по комнатам с помощью простых команд
• Интерактивный инвентарь - сбор и использование предметов
• Загадки и головоломки - каждая комната содержит уникальные испытания
• Система случайных событий - непредсказуемые встречи и находки
• Пошаговый игровой процесс - каждая команда приближает к победе
• Система очков и достижений - отслеживайте свой прогресс

2. Установка и запуск

Предварительные требования:
• Python 3.9 или выше
• Poetry (менеджер зависимостей)

Способ 1: Использование Poetry (рекомендуется!)
# Клонируйте репозиторий
git clone https://github.com/natpavge-cloud/project1_PavlovaNG_M25-555
cd project1_PavlovaNG_M25-555

# Установите зависимости
poetry install

# Активируйте виртуальное окружение
poetry shell

# Запустите игру
poetry run python main.py

Способ 2: Использование Make (если есть Makefile)
# Установка и запуск одной командой
make install
make run

# Или просто запуск
make project

Способ 3: Прямой запуск
# Установка зависимостей вручную
pip install -r requirements.txt

# Запуск игры
python main.py

3. Игровой процесс

Основные команды:
Движение:
  go <направление>  - перейти в направлении (north/south/east/west)
  north/n/север     - перейти на север (краткая форма)
  south/s/юг        - перейти на юг
  west/w/запад      - перейти на запад  
  east/e/восток     - перейти на восток

Взаимодействие:
  look              - осмотреть текущую комнату
  take <предмет>    - поднять предмет
  use <предмет>     - использовать предмет из инвентаря
  inventory         - показать инвентарь
  solve             - решить загадку в комнате
  open              - открыть сундук (в treasure_room)

Информация:
  score             - показать текущий счет и статистику
  help              - показать все доступные команды
  quit              - выйти из игры

4. Цель игры:
Найдите ключ от сокровищ или взломайте сундук кодом, чтобы открыть сундук в комнате сокровищ и победить!

5. Демонстрация игрового процесса

Полный сценарий игры (от запуска до победы)

https://asciinema.org/a/yffxQTFsk1uFlH3r

Сценарий включает:
1. Запуск игры — приветственное сообщение и начальная комната
2. Исследование — перемещение по лабиринту, сбор предметов
3. Решение загадок — взаимодействие с головоломками в разных комнатах
4. Случайные события — встречи с неожиданными ситуациями
5. Поиск ключа — нахождение и использование ключа от сокровищ
6. Финальная битва — открытие сундука и победа

Ключевые моменты демонстрации:
• Успешное перемещение по всем доступным комнатам
• Сбор важных предметов: факел, меч, ключи
• Решение загадок в зале, библиотеке и комнате с ловушками
• Преодоление случайных событий и ловушек
• Использование ключа для доступа к сокровищам
• Открытие сундука и достижение победы

Статистика победного прохождения:
• Время игры: ~15-20 минут
• Количество шагов: 25-35
• Решено загадок: 4-6
• Собрано предметов: 5-7
• Итоговый счет: 150-200 очков
